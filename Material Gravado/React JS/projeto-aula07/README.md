# Projeto Aula 07

Neste exercício deve se adicionar as seguintes funcionalidades ao [projeto base](https://drive.google.com/file/d/1_oOuK5p370tRRVWSaHkNLnsU4wef1g0V/view?usp=sharing):

1. A rota Dashboard deve ser uma rota privada somente acessível para usuários logados.
2. Após logar ou se cadastrar o usuário deve receber um feedback visual de sucesso ou erro, em caso de sucesso, também deve ser redirecionado para a Dashboard.
3. Os dados cadastrados devem ser salvos no **localstorage**, assim como a validação de cadastro será feita verificando o localstorage (má prática porem necessária para não depender de uma API externa).
